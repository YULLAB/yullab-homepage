# YULLAB 공식 홈페이지

대한민국 AI 기술회사 율랩(YULLAB Inc.)의 공식 홈페이지입니다.

## 기술 스택

- Next.js App Router
- React
- Tailwind CSS
- Vercel 배포 대응

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

Vercel에서 이 폴더를 프로젝트 루트로 선택한 뒤 기본 설정 그대로 배포할 수 있습니다.

문의 이메일은 `team@yullab.co.kr`로 설정되어 있습니다.
